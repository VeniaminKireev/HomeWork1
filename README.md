## Результаты работы

EDA
- Проведен анализ данных, выявлены пропуски и выбросы
- Преобразованы признаки (mileage, engine, max_power)
- Удалены дубликаты и ненужные столбцы
- Построены визуализации распределений

Моделирование
- LinearRegression: Базовый подход, R² = 0.5619
- Lasso: С L1-регуляризацией, R² = 0.6146
- Ridge: С L2-регуляризацией, R² = 0.5694
- ElasticNet: Комбинация L1 и L2, R² = 0.5907

Наилучшая модель:

Lasso регрессия показала наилучшие результаты

1. Оптимальный alpha: 100.00
2. R² на валидации: 0.6146
3. Business metric: 0.2252 (доля предсказаний с ошибкой ≤ 10%)
